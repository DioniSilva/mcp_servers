# Obsidian Markdown Skill

Create notes using Obsidian Flavored Markdown.

Use:

- YAML frontmatter for properties such as `title`, `tags`, `aliases`, and `status`.
- Wikilinks for internal references: `[[Note Name]]`.
- Embeds for notes, images, and PDFs: `![[Asset.png]]`.
- Callouts with `> [!type]`.
- Tags either inline (`#topic`) or in frontmatter.

Prefer wikilinks for vault-local references and standard Markdown links for external URLs.
